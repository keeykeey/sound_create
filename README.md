# Sound_Create

1 概要

　・作曲などした自作音源をアップロードし、共有、公開できるサイトです。

    ・url  https://www.keeykeey.com　

2 機能

投稿者

　・会員登録、ログイン機能

　・音声コンテンツのアップロード

　・アップロードしたコンテンツの削除

リスナー（一般ユーザー）

　・気に入った投稿者のお気に入り登録

　・気に入った音声コンテンツへのいいね機能

３ 使用技術

　・django-rest-framework

　・react

　・docker-compose

　・PostgreSQL

　・aws(ec2,route53)

　・nginx
